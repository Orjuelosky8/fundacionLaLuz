# **App Name**: LuzIA

## Core Features:

- AI-Powered Mental Health Assessment: Utilize a generative AI tool, similar to MedByte, to assess user's mental state (stress, anxiety, depression) based on their input and suggest appropriate resources and plans.
- Personalized Wellness Platform: Offer a platform integrating AI, telemedicine, nutrition, and exercise recommendations tailored to individual needs, inspired by Mutuo.
- Substance Use Risk Reduction Tool: Provide an AI-driven tool like DroguIA to offer guidance and resources for substance use risk reduction, targeting both individuals and families.
- Comprehensive Landing Page: Create a landing page with clear sections for target audiences (families, individuals seeking treatment, EPS/allies), trust-building elements (verifiable data, testimonials), and simple service descriptions.
- Service and Location Directory: Develop a directory showcasing the foundation's services with clear, user-friendly descriptions. Each location contains phone numbers and working hours.
- Contact and Consultation Forms: Implement a short and direct form for inquiries, including fields for Name, Phone, Location, and Reason for Contact.
- Financial Transparency Page: Improve the 'Estados Financieros' page by embedding the PDFs directly into the website with an easier access to previous years.

## Style Guidelines:

- Azul principal: #0B5FA5. Uso: identidad, header, botones principales, logo background. Mensaje: confianza, estructura, IPS seria.
- Blanco clínico: #FFFFFF. Uso: texto sobre azul, espacios de respiro. Mensaje: claridad, transparencia.
- Azul secundario: #2F7EC1. Uso: hover, links, secciones secundarias. Complementa al azul principal sin competir.
- Verde apoyo: #5FB3A2. Uso: estados positivos, mensajes de avance, acompañamiento. Mensaje: recuperación, equilibrio, camino.
- Amarillo luz: #F2C94C. Uso: íconos, pequeños highlights, CTA secundarios. Mensaje: luz, posibilidad, “sí se puede”.
- Fondo humano: #F7F9FB. Uso: fondos largos, secciones informativas. Mensaje: descanso visual, cercanía.
- Texto principal: #2D2D2D. Uso: textos largos. Más amable que negro absoluto.
- Headline font: 'Playfair', serif, offering a touch of elegance; Body text font: 'PT Sans', sans-serif, chosen for clarity in longer content.
- Utilize soft, rounded icons representing mental health, addiction treatment, and technology concepts.
- Implement a clear, sectioned layout for the landing page, catering to the personas 'Soy familiar', 'Busco tratamiento', and 'Soy EPS/aliado'.
- Incorporate smooth transitions and subtle animations to enhance user experience and engagement.
